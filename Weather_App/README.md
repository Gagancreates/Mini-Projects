# 🌦️ Weather App  

A simple weather application that fetches real-time weather data using the OpenWeather API.  

## 🚀 Features
- Search weather by city 🌍  
- Displays temperature, humidity, and weather conditions ⛅  
- Clean and responsive UI 🎨  

## 🛠️ Tech Stack
- HTML
- Tailwind CSS  
- OpenWeather API
